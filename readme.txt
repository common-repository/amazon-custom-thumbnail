== Description ==
With Amazon custom thumbnail, you can Create an attractive thumbnail display for your amazon affiliate sites :).
Thank you for using my plugin. for more info visit me on http://seegatesite.com/amazon-custom-thumbnail-plugin-new-version-launch/

== Installation ==

- Upload the Amazon Asin Grabber plugin to your blog, Activate it,this plugin will be replace at settings->amazon custom thumbnail .You're done!
- To change plugin width , go to settings->amazon custom thumbnail and set your width.
- To get custom field value , visit http://seegatesite.com/tools/azon-custom-thumbnail-helper.php
